# Security Policy

## Reporting a Vulnerability

Please report security issues privately through GitHub Security Advisories:

https://github.com/adomorn/gh-alerter/security/advisories/new

Avoid opening public issues for vulnerabilities until a fix is available.

## Scope

Supported security reporting scope:

- GH Alerter source code
- macOS notification and settings behavior
- Release packaging scripts
- GitHub Actions workflows

Out of scope:

- Issues in GitHub CLI itself
- Issues in macOS notification delivery outside this app
- User-specific GitHub account or organization permissions
